Student Academic Management System
=================================

1. Overview
-----------
A Python-based system built using Object-Oriented Programming (OOP) to manage student records, courses, grades, and GPA.

Purpose:
To simulate a real-world academic system and demonstrate OOP concepts and problem-solving skills.


2. Features
-----------
- Add, edit, and delete students
- Search students by ID
- Manage courses and grades
- Automatic GPA calculation
- Display all students
- Identify top students using sorting


3. System Design
----------------
- Person:
  Base class storing general data (name, age)

- Student:
  Inherits from Person
  Stores ID, courses, grades
  Calculates GPA

- Manager:
  Controls system operations:
  Add / Delete / Search / Display students
  Find top-performing students


4. OOP Concepts Used
--------------------
- Encapsulation:
  Data and methods are inside classes

- Inheritance:
  Student inherits from Person

- Abstraction:
  GPA calculation hidden inside method

- Polymorphism:
  Flexible methods like display_info()


5. Test Cases
-------------
- Add student → appears in system
- Add course → GPA updates correctly
- Delete student → removed successfully
- Search student → correct data shown


6. Challenges
-------------
- Designing clean class structure
- Handling empty grades in GPA calculation
- Managing student data efficiently
